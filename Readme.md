## 🚀 Features

- ⚡ Lightweight and fast
- 🖥️ Native Linux terminal support
- ⌨️ Raw keyboard input handling
- 🎛️ Custom terminal control with `termios`
- 📦 Zero external dependencies
- 🧠 Memory-efficient design
- 🔥 Built completely in C
